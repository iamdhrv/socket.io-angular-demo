# socket.io-angular-demo

## Project Setup For Angular

```
npm install
```

## Run Locally

```
npm run || ng serve
```
```bash
URL: http:\\localhost:4200\
```
## Server Setup And Run Locally
```
// install dependency
npm install

//Run Server
npm start
```
